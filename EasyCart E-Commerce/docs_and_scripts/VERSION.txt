========================================
TESTING SERVER DELAYS
========================================

1. Open includes/server-simulator.php
2. Verify SERVER_DELAYS_ENABLED = true
3. Choose profile:
   - FAST: Quick responses (good connection)
   - NORMAL: Realistic delays (default)
   - SLOW: Overloaded server
   - UNRELIABLE: Random delays (bad connection)
4. Test cart operations, wishlist, search
5. Watch skeleton loading animations

========================================
DISABLE DELAYS FOR PRODUCTION
========================================

Set in includes/server-simulator.php:
define('SERVER_DELAYS_ENABLED', false);
